Air Draw 

Draw in the air using just your fingers.

This project is a browser-based drawing tool that uses hand gestures to let users draw, move, and erase strokes without touching the screen. By tracking finger movements through the camera, the application creates a fun and interactive drawing experience.

Features
✋ Draw in the air using finger movement
🤏 Pinch gesture to move objects
🧽 Erase drawings with gesture controls
🎨 Real-time drawing on a virtual canvas
Uses webcam for hand tracking
Demo

How It Works

The application captures video from the user's webcam and detects hand gestures. Finger positions are translated into drawing coordinates on a virtual canvas in the browser.
